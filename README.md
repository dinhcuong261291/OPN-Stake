# OPN Stake

## Liquid Staking & Restaking Infrastructure for OPN Chain

OPN Stake is the foundational staking infrastructure powering the OPN ecosystem.

Inspired by the success of Ethereum liquid staking protocols and next-generation shared security networks, OPN Stake enables users to stake OPN while maintaining liquidity through stOPN and unlocking additional yield opportunities through rstOPN.

---

# Vision

Build the most secure, decentralized and capital-efficient staking layer on OPN Chain.

Our long-term mission is to become the economic security layer of the OPN ecosystem.

---

# Core Products

## Liquid Staking

Users deposit OPN into the protocol and receive stOPN.

Benefits:

* Maintain liquidity
* Earn staking rewards
* Participate in DeFi
* No lockup periods

---

## Liquid Restaking

Users can restake stOPN to secure additional services.

Examples:

* Data Availability Layer
* Oracle Networks
* Cross-chain Messaging
* AI Inference Networks
* Decentralized Sequencers

Rewards are generated from both staking and restaking activities.

# How OPN Stake Works

1. User deposits OPN

2. Protocol delegates OPN to validators

3. User receives stOPN

4. Rewards accumulate automatically

5. stOPN can be used in DeFi

6. User optionally restakes stOPN

7. Additional rewards are generated
